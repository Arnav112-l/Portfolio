# Arnav Singh — Portfolio

Single-page founder portfolio. Edit all content in **`src/content.ts`**.

## Run

```bash
npm install
npm run dev
```

## StayID screenshots

Place PNGs in `public/stayid/`:

- `landing.png`
- `dashboard.png`
- `admin.png`

## Deploy

```bash
npm run build
```

Deploy the `dist/` folder to Vercel, Netlify, or GitHub Pages.
